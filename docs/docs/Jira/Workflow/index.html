Workflow 工作流

transition

  • 触发器（Triggers ）： 与 Atlassian 的其他产品之间互通
  • 触发条件（Conditions） ： 不满足条件，则指定的按钮不显示
  • 校验条件（Validator）： 不满足条件，显示提示信息
  • 后置处理器（Post Function）： 状态变更后执行的操作
